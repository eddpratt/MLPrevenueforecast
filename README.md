# MLP Molecular Testing Revenue Forecast Calculator

An interactive web calculator for projecting molecular laboratory testing revenues based on daily plate volumes and Medicare reimbursement rates.

## Features

- 11 test panel types (Covid/Flu/RSV, RPP Lite, RPP, RPP Plus, UTI Lite, UTI, UTI Plus, STI, HPV, Vaginitis, Wound)
- Adjustable working days per month
- Real-time monthly and annual revenue projections
- Net margin calculations
- CSV export functionality
- Professional medical/lab aesthetic

## Usage

1. Enter daily plate values for each test panel you plan to run
2. Adjust working days per month (default: 21)
3. View real-time calculations for monthly and annual projections
4. Export results to CSV for further analysis

## Technology

- React 18
- No external dependencies required
- Pure CSS styling

## Installation

```bash
# Clone the repository
git clone https://github.com/YOUR_USERNAME/mlp-calculator.git

# Navigate to project directory
cd mlp-calculator

# Install dependencies
npm install

# Start development server
npm start
```
